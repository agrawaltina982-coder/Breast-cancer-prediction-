# Breast-cancer-prediction-
# 🩺 Breast Cancer Prediction Using Machine Learning

## Overview

This project develops a Machine Learning model to predict whether a breast tumor is **Malignant** or **Benign** based on medical diagnostic features. The model is trained using the Breast Cancer Wisconsin Dataset and can assist in the early detection of breast cancer.

## Key Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Scaling
* Machine Learning Model Training
* Model Evaluation and Accuracy Analysis
* Predictive System for New Patient Data

## Technologies Used

* Python
* Google Colab
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Dataset

The project uses the Breast Cancer Wisconsin Dataset, which contains diagnostic measurements of breast tumors. The target variable classifies tumors as:

* **Benign (B)** – Non-cancerous
* **Malignant (M)** – Cancerous

## Machine Learning Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Model Training
6. Model Evaluation
7. Prediction System Development

## Results

The trained model successfully classifies breast tumors as benign or malignant with high accuracy. The prediction system allows users to input feature values and receive an instant diagnosis prediction.

## How to Run

1. Open the notebook in Google Colab.
2. Upload the dataset file.
3. Run all cells sequentially.
4. Enter sample feature values in the prediction section.
5. View the predicted result.
